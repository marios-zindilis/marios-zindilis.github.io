syslog priorities

A list of syslog priorities, sorted by importance, ascending:

  1. debug
  2. info
  3. notice
  4. warning (previously warn, now deprecated)
  5. err (previously error, now deprecated)
  6. crit
  7. alert
  8. emerg (previously panic, now deprecated)
